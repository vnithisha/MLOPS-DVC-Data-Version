1. Create git repo and clone it in local
2. Create code.py and add code to it(it will save a csv file to a new data folder)
3. Do a git add-commit-push before initializing dvc(git add . , git commit -m "commit message", git push origin main")
# pip install dvc
4. Now initialize dvc ("dvc init" (creates .dvcignore, .dvc)
5. Now do "mkdir S3" (create a new S3 directory)
6. Now we do "dvc remote add -d myremote S3"
7. Next "dvc add data/"
